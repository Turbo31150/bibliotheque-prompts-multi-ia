# Multi-IA -- Web & Social

## Description

Prompts pour orchestrer plusieurs IA dans la creation de contenu web et la gestion des reseaux sociaux : ideation, redaction, optimisation SEO et analyse de performance.

## Cas d'usage
- Pipeline de creation de contenu multi-IA
- SEO optimise par analyse croisee
- Veille concurrentielle multi-sources
- Gestion de communaute assistee
- Strategie sociale multi-plateforme

---

## Prompts prets a copier

### 1 -- Pipeline de creation de contenu

```
## Perplexity (recherche)
"Recherche les tendances actuelles sur [SUJET]. Top articles, discussions, questions frequentes."

## ChatGPT (redaction)
"A partir de ces tendances, redige un article de blog de 1500 mots sur [SUJET].
Ton : [STYLE]. Structure SEO avec H1/H2/H3."

## Claude (revue)
"Revois cet article : exactitude technique, clarte, structure, engagement.
Propose des ameliorations specifiques."

## Gemini (adaptation multi-plateforme)
"Adapte cet article en :
- Thread Twitter (5 tweets)
- Post LinkedIn (500 mots)
- Description YouTube (300 mots)
- Post Reddit (format communautaire)"
```

---

### 2 -- Optimisation SEO multi-IA

```
## Perplexity
"Recherche les mots-cles les plus recherches autour de [SUJET].
Volume de recherche, difficulte, questions People Also Ask."

## ChatGPT
"Optimise cet article [COLLER] avec ces mots-cles [LISTE].
Densite naturelle, title tag, meta description, alt texts."

## Claude
"Revois l'optimisation SEO. Verifie :
- Placement naturel des mots-cles
- Cannibalisation avec d'autres pages
- Schema markup adapte
- Liens internes pertinents"
```

---

### 3 -- Veille concurrentielle

```
## Perplexity
"Analyse la presence en ligne de [CONCURRENT] : site, reseaux sociaux, contenu recent."

## ChatGPT
"Compare leur strategie de contenu avec la notre :
- Themes couverts
- Frequence de publication
- Formats utilises
- Engagement moyen"

## Claude
"Identifie les opportunites :
- Sujets que le concurrent ne couvre pas
- Formats sous-exploites
- Segments d'audience negliges
- Recommandations strategiques"
```

---

### 4 -- Gestion de crise multi-IA

```
## Claude (analyse)
"Analyse cette situation de crise [DESCRIPTION].
Gravite, audience touchee, timeline, risques d'escalade."

## ChatGPT (redaction)
"Redige les reponses adaptees :
- Message public (transparent, empathique)
- Reponses aux commentaires (templates par type)
- FAQ de crise"

## Perplexity (contexte)
"Recherche des crises similaires et comment elles ont ete gerees.
Best practices de communication de crise."

## Claude (strategie)
"Plan de sortie de crise :
- Actions immediates
- Communication J+1, J+3, J+7
- Mesures de reconstruction de confiance"
```

---

### 5 -- Calendrier editorial multi-IA

```
## Perplexity
"Evenements, lancements et dates cles pour [SECTEUR] sur les 3 prochains mois."

## ChatGPT
"Cree un calendrier editorial de 3 mois :
- 3 posts/semaine
- Mix de formats (article, video, infographie)
- Aligne sur les evenements trouves"

## Claude
"Revois le calendrier :
- Equilibre des themes
- Coherence avec la strategie globale
- Moments forts a ne pas manquer
- Suggestions d'amelioration"
```

---

## Exemples d'utilisation

### Exemple : Article optimise
**Workflow** : Perplexity (mots-cles) → ChatGPT (redaction) → Claude (revue) → Gemini (adaptation)

**Resultat attendu** : Article SEO-optimise avec versions pour chaque plateforme.

---

## Effet sur le modele
- La recherche Perplexity fournit des donnees reelles pour la creation
- ChatGPT excelle dans la redaction creative
- Claude ajoute la rigueur dans la revue
- L'adaptation multi-plateforme couvre tous les canaux
