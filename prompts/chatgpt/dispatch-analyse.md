# ChatGPT — Prompts d'Analyse & Generation

> Acces : via CDP (Chrome 9222) ou via `jai chatgpt "prompt"`

---

## Prompt — Analyse de code avec vision

```
Analyse cette architecture logicielle. Identifie les patterns, les anti-patterns, les risques de scalabilite. Propose 3 ameliorations concretes classees par impact.
```

## Prompt — Generation de contenu marketing

```
Genere un post LinkedIn de 200 mots sur [SUJET]. Ton : professionnel mais accessible. Hook fort en premiere ligne. 3 bullet points de valeur. CTA final. Hashtags pertinents.
```

## Prompt — Brainstorming produit

```
Je suis freelance IA specialise en [DOMAINE]. Propose 5 offres de service innovantes que je ne propose pas encore, avec pour chacune : cible, prix suggere, argument cle, et temps de livraison estime.
```

## Prompt — Review de proposition commerciale

```
Voici ma proposition commerciale pour [CLIENT]. Note chaque section /10 : accroche, comprehension du besoin, solution proposee, tarif, argumentaire. Ameliore les sections < 7/10.
```
