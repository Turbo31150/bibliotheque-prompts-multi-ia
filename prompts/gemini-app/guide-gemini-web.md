# Gemini (application web) — Guide complet

> Google Gemini dans le navigateur : prompts, Gems, extensions et bonnes pratiques.

---

## Accès

- URL : [gemini.google.com](https://gemini.google.com)
- Compte Google requis (gratuit ou Google One AI Premium).

## Modèles

| Modèle | Disponibilité | Usage |
|---|---|---|
| **Gemini 2.5 Pro** | Premium | Le plus capable. Raisonnement, code, analyse. |
| **Gemini 2.5 Flash** | Gratuit | Rapide, bon pour les tâches courantes. |

## Gems (agents personnalisés)

Les Gems sont des agents spécialisés avec des instructions persistantes.

### Créer un Gem

1. Aller dans *Gem Manager*.
2. Cliquer *New Gem*.
3. Définir le nom, les instructions et le comportement.

### Exemples de Gems utiles

| Gem | Instructions |
|---|---|
| **Code Reviewer** | « Tu es un expert en review de code. Vérifie la qualité, la sécurité et les performances. Donne un score sur 10. » |
| **Traducteur Tech** | « Traduis le texte technique en français. Garde les termes anglais pour les concepts sans équivalent. » |
| **Résumeur** | « Résume en 3 bullet points maximum. Commence par le plus important. » |

## Extensions

Les extensions connectent Gemini à vos données :
- **Google Workspace** : accès aux emails, docs, drive.
- **Google Maps** : localisation et itinéraires.
- **YouTube** : résumé de vidéos.
- **Google Flights/Hotels** : recherche de voyages.

## Bonnes pratiques

1. **Uploader des fichiers** : PDF, images, code — Gemini les analyse nativement.
2. **Utiliser les Gems** pour les tâches récurrentes plutôt que de réécrire les instructions.
3. **Comparer les réponses** : demander « Compare avec une approche alternative ».
4. **Demander des sources** : « Cite tes sources » pour vérifier les informations.
