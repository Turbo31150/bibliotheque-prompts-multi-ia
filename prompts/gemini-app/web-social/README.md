# Gemini App -- Web & Social

## Description

Prompts pour utiliser Gemini App dans le marketing digital, la creation de contenu web et l'analyse de reseaux sociaux. Gemini App est ideal pour la generation de contenu creatif et l'analyse de tendances.

## Cas d'usage
- Creation de contenu pour reseaux sociaux
- Strategie de marketing digital
- Analyse de tendances et audiences
- SEO et optimisation de contenu
- Community management

---

## Prompts prets a copier

### 1 -- Creer une strategie de contenu

```
Cree une strategie de contenu pour [SUJET/MARQUE] sur [PLATEFORME] :

## OBJECTIF : [NOTORIETE / ENGAGEMENT / CONVERSION]
## AUDIENCE : [DESCRIPTION DE LA CIBLE]
## FREQUENCE : [X posts/semaine]

## STRATEGIE
1. PILIERS DE CONTENU (3-5 themes recurrents)
   - Theme, proportion, format prefere

2. CALENDRIER EDITORIAL (1 mois)
   - Jour par jour : type de post, theme, format
   - Heures de publication optimales

3. TEMPLATES DE POSTS (1 par pilier)
   - Structure
   - Hooks d'accroche (5 variantes)
   - CTA (call to action)
   - Hashtags

4. METRIQUES DE SUCCES
   - KPIs par pilier
   - Objectifs chiffres a 1 mois et 3 mois
   - Outils de mesure
```

---

### 2 -- Optimiser le SEO d'un contenu

```
Optimise ce contenu pour le SEO :

[COLLER LE CONTENU]

## OPTIMISATIONS
1. MOTS-CLES
   - Mot-cle principal et secondaires
   - Densite recommandee
   - Placement (titre, H2, intro, conclusion)

2. STRUCTURE
   - Title tag optimise (< 60 caracteres)
   - Meta description (< 160 caracteres)
   - Structure H1/H2/H3 avec mots-cles
   - Liens internes suggeres

3. CONTENU
   - Sections manquantes pour couvrir le sujet completement
   - Questions "People Also Ask" a traiter
   - Featured snippet potential (format liste/tableau/definition)

4. TECHNIQUE
   - Schema markup recommande (JSON-LD)
   - Alt texts pour les images
   - URL optimisee
```

### 3 -- Analyser les tendances d'un secteur

```
Analyse les tendances actuelles dans le secteur [SECTEUR] :

1. TENDANCES MACRO (3-5)
   - Description de la tendance
   - Depuis quand / stade d'adoption
   - Impact sur le secteur

2. TENDANCES CONTENU
   - Formats qui marchent (video, article, infographie...)
   - Sujets les plus engageants
   - Ton et style preferes par l'audience

3. CONCURRENCE
   - Top 5 des acteurs les plus visibles
   - Ce qu'ils font bien
   - Opportunites qu'ils manquent

4. RECOMMANDATIONS
   - 5 sujets de contenu a forte probabilite de viralite
   - Formats recommandes pour chacun
   - Timing optimal (saisonnalite, evenements)
```

---

### 4 -- Generer des posts multi-plateformes

```
A partir de ce sujet : [DESCRIPTION DU SUJET]

Genere des posts adaptes pour chaque plateforme :

## TWITTER/X
- Thread de 5 tweets
- Premier tweet = hook viral
- Dernier tweet = CTA
- Hashtags pertinents

## LINKEDIN
- Post long format (1500 caracteres)
- Ton professionnel mais accessible
- Structure : question → experience → lecon → CTA

## YOUTUBE (description)
- Titre accrocheur (< 60 caracteres)
- Description SEO (500 mots)
- Tags (20 minimum)
- Timestamps suggeres

## BLOG
- Article complet (1000 mots)
- Structure SEO (H1, H2, H3)
- Meta description
- 3 images suggerees avec alt text
```

---

### 5 -- Gerer une crise sur les reseaux sociaux

```
Aide-moi a gerer cette situation de crise :

## SITUATION
[DECRIRE LA CRISE : commentaires negatifs, bad buzz, erreur publique...]

## PLAN DE GESTION DE CRISE
1. EVALUATION (gravite 1-5, viralite, impact)
2. REPONSE IMMEDIATE (< 1 heure)
   - Message type (empathique, factuel, responsable)
   - Qui repond (personne, ton, canal)
3. SUIVI (24-48h)
   - Messages de mise a jour
   - Reponses aux commentaires individuels
   - Monitoring des mentions
4. RESOLUTION
   - Actions correctives concretes
   - Communication de resolution
5. POST-CRISE
   - Bilan et lecons apprises
   - Ajustement de la strategie
   - Contenu de reconstruction de confiance
```

## Exemples d'utilisation

### Exemple : Contenu tech
**Prompt** : "Genere 5 posts LinkedIn sur le theme 'auto-hebergement vs cloud' pour un public DevOps."

**Resultat attendu** : 5 posts structures avec hooks, arguments et CTA, prets a publier.

### Exemple : SEO
**Prompt** : "Optimise cet article sur 'installer Ollama sur Linux' pour le SEO [coller]."

**Resultat attendu** : Article restructure avec mots-cles, meta tags et schema markup.

## Effet sur le modele
- Gemini App genere du contenu creatif et adapte au ton de chaque plateforme
- Les prompts multi-plateformes exploitent la capacite a adapter le style
- La demande de KPIs chiffres produit des strategies mesurables
- Le contexte d'audience precise ameliore la pertinence du contenu
