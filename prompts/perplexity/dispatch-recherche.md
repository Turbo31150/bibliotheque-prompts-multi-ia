# Perplexity — Prompts de Recherche & Verification

> Utiliser Perplexity comme moteur de recherche IA avec sources.
> Acces : via CDP (Chrome 9222) ou via `jai perplexity "prompt"`

---

## Prompt — Recherche marche freelance

```
Tarifs freelance IA France 2026 : quels sont les TJM moyens par niveau (debutant, confirme, senior, expert) sur Codeur.com et Malt ? Quels USP marchent le mieux pour les agents autonomes et la voice AI ?
```

## Prompt — Analyse concurrentielle

```
Analyse les profils freelance IA les mieux notes sur [PLATEFORME] France. Quels arguments, realisations, tarifs affichent-ils ? Compare avec un profil proposant [MES_COMPETENCES]. Forces et faiblesses vs concurrence ?
```

## Prompt — Verification de contenu avant publication

```
Verifie ce contenu pour [PLATEFORME]. Note la credibilite /10, ameliore la structure visuelle, corrige le wording. Propose une version amelioree avec meilleur impact. Contenu : [CONTENU]
```

## Prompt — Veille technologique

```
Quelles sont les dernieres avancees en [SUJET_TECH] cette semaine ? Sources fiables uniquement. Resume en 5 bullet points avec liens.
```

## Prompt — Recherche client/prospect

```
Analyse le site [URL_CLIENT]. Quels sont leurs besoins en IA ? Comment puis-je positionner mes services (agents autonomes, voice AI, automatisation) pour leur etre utile ? Propose un angle d'approche.
```
